# Knowledge Daily - AI 生成提示词模板

## 使用说明

每次生成内容时，将以下模板填充后发送给 AI（Claude/GPT等），即可生成符合格式的学习内容。

---

## 基础提示词（每次使用前置）

```
你是一位资深的知识教师，擅长用知乎回答风格讲解知识：故事引入、深入分析、金句收尾。

目标用户是：游戏设计/互联网从业者
习题风格：结合游戏设计、互联网产品、用户体验等实际应用场景

输出要求：
1. 深入分析部分需要结合大量实际案例，让知识更形象
2. 重点记忆部分用表格/口诀/对比等方式便于记忆
3. 习题必须是应用型，结合游戏/互联网行业
4. 每道习题的答案要详细展开分析，不惜篇幅
5. 如果与之前学过的知识有关联，需要在"知识关联"部分指出
```

---

## 单学科生成模板

```
请为以下学科生成第 X 期内容：

【学科】：{subject_name}
【章节】：{chapter_title}
【本期主题】：{episode_title}
【期数】：第 {current}/{total} 期

【已学过的相关知识】：
{previous_knowledge}

请按以下结构输出：

## {icon} {subject_name} | 第{current}/{total}期
### {episode_title}

#### 🔗 知识关联（Callback）
[回顾之前学过的相关知识，指出本期如何关联]

#### 引入故事
[用一个真实案例、生活场景或有趣的问题引入，300字左右]

#### 核心概念
[一句话定义核心概念]

#### 深入分析
[详细讲解，包含多个实际案例、游戏/互联网行业类比，800-1200字]

#### 重点记忆
[用表格、口诀、对比等方式总结要点]

#### 一句话带走
[金句收尾]

#### 📝 课后习题（游戏/互联网应用向）
[5道应用型习题，只写题干]

#### 参考答案
[5道题的详细解析，每题200-400字]

#### 知识存档JSON
[输出本期知识点的JSON结构]
```

## 多学科批量生成模板

```
请为今日学习生成以下学科内容：

【全局期数】：第 {global_episode} 期
【日期】：{date}

---

【学科1】：{subject1_name}
- 章节：{chapter1_title}
- 本期主题：{episode1_title}  
- 期数：第 {current1}/{total1} 期
- 已学相关知识：{previous1}

【学科2】：{subject2_name}
- 章节：{chapter2_title}
- 本期主题：{episode2_title}
- 期数：第 {current2}/{total2} 期
- 已学相关知识：{previous2}

---

请按模板格式输出每个学科的完整内容，并在最后输出：
1. 今日学习统计表格
2. 知识脉络图（ASCII）
3. 完整的知识存档JSON
```

## 知识总结生成模板

```
请根据以下已学知识库，生成知识脉络总结：

【学科】：{subject_name}
【已完成期数】：第 1-{current} 期
【知识库内容】：
{knowledge_base_json}

请输出：
1. 知识脉络图（展示各知识点之间的关联）
2. 核心概念清单（按重要程度排序）
3. 跨学科关联分析（与其他学科的联系）
4. 复习建议（哪些知识点需要重点回顾）
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| {subject_name} | 学科名称 | 宏观经济学 |
| {icon} | 学科图标 | 📈 |
| {chapter_title} | 章节标题 | 货币政策 |
| {episode_title} | 本期主题 | 央行如何调控经济 |
| {current} | 当前期数 | 3 |
| {total} | 总期数 | 30 |
| {previous_knowledge} | 已学相关知识摘要 | 第1期学了通胀，第2期学了货币乘数... |
| {global_episode} | 全局期数 | 15 |
| {date} | 日期 | 2026年2月22日 |
