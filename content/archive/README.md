# 📚 邮件存档目录

本目录用于存放已发送的每日知识邮件历史记录。

## 命名规范

```
episode_001_2026-02-18.md  # 第1期_发送日期
episode_002_2026-02-19.md  # 第2期_发送日期
...
```

## 目录结构

```
archive/
├── README.md           # 本说明文件
├── episode_001_*.md    # 第1期存档
├── episode_002_*.md    # 第2期存档
└── ...
```

## 注意事项

- 每次发送邮件后，`send_email.py` 会自动将内容存档到此目录
- 存档文件为只读历史记录，请勿修改
- 如需重新发送历史邮件，可从此处复制内容到 `pending_email.md`
